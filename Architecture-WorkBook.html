Key Components Explained

1. Data Module

Manages the raw data arrays and parameter values from ξ variables. Handles data sorting and extraction.

2. Matrix Operations

Core mathematical operations using math.js library. Creates Vandermonde matrices and performs linear algebra.

3. Statistical Solvers

Implements OLS and Ridge regression algorithms. Handles numerical stability issues with extreme data values.

4. Prediction Engine

Generates predictions from fitted models and calculates performance metrics (MSE, R², MAE).

5. Bayesian Analysis

Implements Gamma distribution functions and posterior calculations for Section 7.

6. Visualization Module

Creates all plots using Plotly.js. Handles different chart types for each section.

7. DOM Controller

Manages HTML element updates, table creation, and dynamic content insertion.

Implementation Notes

Error Handling

  • OLS solver includes try-catch for numerical instability
  • Ridge regression provides fallback for extreme data values
  • Gamma function uses Stirling's approximation for large values

Performance Optimizations

  • Matrix operations use efficient math.js implementations
  • Plots are rendered asynchronously after DOM ready
  • Data is pre-sorted for visualization efficiency

External Dependencies

  • Plotly.js: Interactive plotting library
  • math.js: Matrix and numerical operations
  • MathJax: LaTeX equation rendering
